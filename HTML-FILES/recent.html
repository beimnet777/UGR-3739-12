Project 1
This project will be much like the to-do list project. The aim here is to design and build a notes app that can take multiple entries per note. It allow users to select a note when they launch the app. When they choose a note, a new entry will be automatically tagged along with the current date, time, and location. Users can also sort and filter their entries based on this metadata
